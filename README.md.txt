# Finance-Nerdery Project

## Overview

Welcome to the **Finance-Nerdery Project**, a comprehensive framework designed to explore, analyze, and test financial trading strategies using Python. The project supports modularity, extensibility, and robust error handling to facilitate data analysis, backtesting, and trading strategy development.

## Project Features

- **Data Handling**: Fetches, cleans, and validates financial data from sources like Yahoo Finance, FRED, and SQLite databases.
- **Metrics Calculation**: Computes key financial metrics such as CAGR, Sharpe Ratio, Sortino Ratio, and Maximum Drawdown.
- **Strategy Simulation**: Implements and tests trading strategies such as Buy-and-Hold and Moving Average Crossover.
- **Output Generation**: Creates structured JSON output for easy integration with downstream tools.
- **Unit Testing**: Includes comprehensive tests for robustness and accuracy.
